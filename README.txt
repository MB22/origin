	README :
			Pour commencer, il faut ouvrir le fichier start.html qui se trouve dans le dossier RPG.
			
			Avancée du projet à l'heure actuelle :
				L'index regroupe toutes la structure du site. Il y aura un inventaire, une carte, un historique, etc...
				
				Ce sur quoi nous avons voulu travailler reste la mise en forme du chapitre d'introduction : les contraintes techniques
et temporelles ne nous permettaient pas, en plus des autres projets, de réaliser le jeu dans son intégralité. Son intérêt est de quelque
peu limité pour le moment mais nous servira de base pour l'avenir, notamment la fonction update, sur laquelle va se reposer tous les contenus des différentes
divisions du site. 

				Nous voulons continuer à mettre en place le contenu du jeu (histoire, carte, bestiaire, etc...) puis de réfléchir aux moyens de l'incorporer au site.
				L'image intéractif en haut à droite du site, qui permet de naviguer entre les différents menu a été réalisé par Matthieu, la dimension graphique a été réalisée "à la main".
